# Annual Report Table Schema (v1)

本文件是“政务公开政府信息公开工作年度报告（全国统一模板）”三张核心表格的**逻辑结构**定义，用于指导 PDF 解析阶段做模板化结构化抽取，并为差异比对与 DOCX 导出提供稳定的数据结构。

## 总体原则

- 目标不是“把 PDF 里的表格画出来”，而是把三张表抽取为**固定的行/列键值结构**（canonical schema）。
- 解析应优先使用 **锚点标题 + 版面坐标聚类/列边界推断** 的方式回填到 schema；遇到跨行/断表/跨页时允许降级为 warnings，但应尽量回填。
- 解析后的结构化结果应保留：`value`、`rawText`（可选）、`confidence`（可选）、`sourceSpan`（可选：页码/坐标范围）。
