# 快速启动指南 v2.0

**版本**: 2.0（后端独立管理）  
**最后更新**: 2025-12-13

---

## 🚀 5 分钟快速开始

### 第一步：编译

```bash
npm run build
```

### 第二步：启动后端

```bash
node dist/index-mock.js
```

后端会在 http://localhost:3000 启动

### 第三步：启动前端

```bash
cd frontend
PORT=3001 npm start
```

前端会在 http://localhost:3001 启动

---

## 📍 访问应用

### 用户端
- **地址**: http://localhost:3001
- **功能**: 创建任务、查看结果

### 后台管理
- **地址**: http://localhost:3000/admin
- **功能**: 管理地区、上传年报、查看记录

---

## 📋 常见操作

### 操作 1：添加新城市

1. 打开 http://localhost:3000/admin
2. 点击 **🗺️ 地区管理**
3. 点击 **➕ 新增地区**
4. 填写信息并保存

### 操作 2：上传年报

1. 打开 http://localhost:3000/admin
2. 点击 **📊 年报汇总**
3. 点击 **⬆️ 上传年报**
4. 选择地区、输入年份、选择文件
5. 点击上传

### 操作 3：创建任务

1. 打开 http://localhost:3001
2. 点击 **➕ 创建任务**
3. 选择城市和年份
4. 点击创建

### 操作 4：查看比对记录

1. 打开 http://localhost:3000/admin
2. 点击 **📋 比对记录**
3. 查看所有比对任务
