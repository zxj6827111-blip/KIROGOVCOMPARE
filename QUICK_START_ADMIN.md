# 后台管理快速启动指南

**最后更新**: 2025-12-13

---

## 🚀 5 分钟快速开始

### 第一步：启动系统

```bash
# 终端 1：启动后端
node dist/index-mock.js

# 终端 2：启动前端
cd frontend
PORT=3001 npm start
```

### 第二步：访问后台

1. 打开浏览器：http://localhost:3001
2. 点击右上角 **🔐 后台管理** 按钮
3. 进入后台管理界面

---

## 📋 三大功能模块

### 1️⃣ 任务管理 (📋)

**查看所有对比任务**

```
左侧：任务列表
右侧：任务详情

操作：
- 选择任务查看详情
- 按状态筛选
- 查看对比结果
```

**状态说明**：
- ⏳ 待处理 - 等待处理
- ⚙️ 处理中 - 正在处理
- ✅ 已完成 - 可查看结果
- ❌ 失败 - 处理失败

---

### 2️⃣ 年报汇总 (📊)

**管理所有上传的年报**

```
操作：
1. 筛选年报
   - 按地区筛选
   - 按年份筛选
   - 按状态筛选
   - 按关键词搜索

2. 修改状态
   - 点击状态下拉框
   - 选择新状态
   - 自动保存

3. 删除年报
   - 点击删除按钮
   - 确认删除
```

**关键规则**：
- ✅ 可用 - 出现在用户侧
- ❌ 禁用 - 不出现在用户侧
- ⚠️ 未分配 - 未分配到地区

---

### 3️⃣ 地区管理 (🗺️)

**管理城市和地区**

```
支持 4 级分类：
- 省份 (Province)
- 地级市 (City)
- 县级市 (District)
- 街道乡镇 (Township)

操作：
1. 新增地区
   - 点击"新增地区"
   - 填写信息
   - 保存

2. 编辑地区
   - 点击"编辑"
   - 修改信息
   - 保存

3. 删除地区
   - 点击"删除"
   - 确认删除
```

## 💡 常见操作

### 操作 1：上传新城市的年报

```
步骤：
1. 后台 → 地区管理
2. 点击"新增地区"
3. 填写：
   - 地区 ID: shenzhen_city
   - 名称: 深圳市
   - 级别: 地级市
   - 父级: guangdong_prov
4. 保存
5. 后台 → 年报汇总 → 上传年报
6. 用户侧自动显示该城市
```

### 操作 2：禁用某份年报

```
步骤：
1. 后台 → 年报汇总
2. 找到要禁用的年报
3. 点击状态下拉框
4. 选择"禁用"
5. 用户侧该年份自动消失
```

### 操作 3：查看任务详情

```
步骤：
1. 后台 → 任务管理
2. 左侧选择任务
3. 右侧显示详情
4. 如果完成，点击"查看对比结果"
```

## 🆘 常见问题

### Q: 如何返回用户端？
A: 点击导航栏的 **👈 返回用户端** 按钮

### Q: 如何新增地区？
A: 地区管理 → 点击 **➕ 新增地区** → 填写信息 → 保存

### Q: 如何修改年报状态？
A: 年报汇总 → 点击状态下拉框 → 选择新状态

### Q: 删除年报后能恢复吗？
A: 不能，删除是永久操作

### Q: 为什么用户侧看不到某个城市？
A: 该城市可能没有 usable 状态的年报

### Q: 如何查看任务的对比结果？
A: 任务管理 → 选择任务 → 点击 **👁️ 查看对比结果**
